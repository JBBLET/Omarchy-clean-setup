# Official repo packages (installed via pacman)
# One per line. Lines starting with # are comments.
#
# We use a "belt-and-braces" approach: packages that Omarchy already ships
# in `omarchy-base.packages` or `omarchy-other.packages` are still listed
# here if we care about them, because `omarchy-remove-preinstalls` (the
# Omarchy menu entry "Uninstall preinstalls") strips a bunch of them. With
# --needed, re-listing is a no-op on a truly fresh install but resurrects
# them on a stripped one. See ~/.local/share/omarchy/install/ for what the
# distro ships, and `omarchy-remove-preinstalls` for what it drops.
#
# Language runtimes (Python, Java, Node, ...) are NOT installed via pacman.
# Omarchy uses mise for runtime version management — see install.sh phase 4.

# Build tools — gradle/cmake/kotlin compilers; mise's java covers the JDK.
gradle
kotlin
cmake
git-lfs

# Belt-and-braces: all in omarchy-base but stripped by omarchy-remove-preinstalls.

# [omarchy] pacman repo that Omarchy adds to /etc/pacman.conf. The others are
# in the regular [extra] repo.

lazydocker
claude-code
spotify
libreoffice-fresh
obsidian
